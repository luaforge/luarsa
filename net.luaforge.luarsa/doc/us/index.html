Overview

LuaRSA is a partial binding for the XySSL library and offers public key cryptographic facilities for Lua 5.1: RSA encryption/decryption using assymetric keys, key generation and digest generation, based on SHA1.

LuaRSA is free software and uses the same license as Lua.

Status

LuaRSA version 0.1.0 for Lua 5.1 is now available for download.

Download

LuaRSA can be downloaded in source code from its LuaForge page. If you are using LuaBinaries 5.1.x, a Windows precompiled version of LuaRSA can also be found at the same LuaForge page. LuaRSA has been tested with Lua 5.1.2.
